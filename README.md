# Installation

pull the monitor's source code

```
git clone https://github.com/coolcode/monitor_docker

cd monitor_docker
```

setup monitor's configuration. Sample: ./setup.sh 127.0.0.1 5

```
./setup.sh {ip} {number of nodes}
```

run monitor

```
docker-compose up -d
```
